# Skill: Audit

Use this skill when reviewing the consistency of this workspace.

## Default Behavior

- Check whether files in `materials/` are reflected in `indexes/INDEX.md`.
- Check whether chapter notes are reflected in the note inventory.
- Check whether chapter notes reference source IDs.
- Check whether exports are missing or stale.

## Repair Policy

- Suggest repairs first.
- Do not rewrite files automatically unless the user asks for repair work.
