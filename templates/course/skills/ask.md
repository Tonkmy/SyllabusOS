# Skill: Ask

Use this skill when answering academic course questions.

## Default Behavior

- Read chapter notes first.
- Read raw materials only when notes and index do not cover the request.
- State whether the answer relies mainly on notes or raw materials.
- Cite source IDs whenever possible.

## Good Outcomes

- The answer is grounded in the course workspace.
- Gaps are stated clearly instead of guessed over.
