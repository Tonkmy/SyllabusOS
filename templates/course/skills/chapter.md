# Skill: Chapter

Use this skill when creating or updating canonical chapter notes for this course workspace.

## Default Behavior

- Prefer updating an existing chapter note over creating a duplicate.
- Keep note filenames stable and meaningful.
- Record source IDs used for the note.
- Reflect the new or updated note in `indexes/INDEX.md`.

## Good Outcomes

- One canonical note per topic cluster when practical.
- Notes become the default retrieval layer for future Q&A.
