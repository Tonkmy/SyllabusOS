# Skill: Intake

Use this skill when processing new files for this workspace.

## Default Behavior

- Review new files from `materials/inbox/`.
- Classify them into `lectures/`, `assignments/`, or `references/`.
- Give new sources simple stable IDs such as `S001`.
- Update `indexes/INDEX.md` after intake.

## Good Outcomes

- The inbox does not stay as a long-term storage area.
- The index reflects what is actually available on disk.
